# symian

> A tiny (927B) CLI wrapper for fs.symlink

## Install

```
$ yarn add symian
```

Or use via npx

```
$ npx symian <target> <path>
```

## Usage

```
  Description
    Creates a link called "path" point to "target".

  Usage
    $ symian <target> <path> [options]

  Options
    -v, --version    Displays current version
    -h, --help       Displays this message
```
